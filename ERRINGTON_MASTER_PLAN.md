# Errington Gardening Encyclopedia Master Plan

## Final Consolidated Specification

This file is the authoritative implementation blueprint for continuing the Errington Gardening Encyclopedia project.

## 1. Project Purpose

Create a scientifically accurate, region-specific, print-ready gardening encyclopedia for Errington, BC, Central Vancouver Island, localized to Zone 8a and nearby 8b microclimates.

The encyclopedia must integrate:

1. Verified climate data.
2. Verified frost dates.
3. Verified botanical taxonomy.
4. Verified sowing, transplant, bloom, and harvest windows.
5. Verified plant morphology.
6. Verified horticultural practices.
7. Per-plant illustration briefs.
8. Per-plant illustration prompts.

## 2. Climate And Zone Baseline

Authoritative local baseline from the direct Errington calendar source:

1. Zone: 8a, with 8b microclimate considerations.
2. Last frost: April 15.
3. First frost: October 15.
4. Frost-free days: approximately 183.
5. Annual rainfall: approximately 1,200 mm.
6. Winter low: rarely below -12 C.
7. Seasonal pattern: wet winters, dry July and August, long daylight summers, maritime influence.

These values override generic external gardening sources when timing conflicts arise.

## 7. Universal Illustration Template

Every illustration must show:

1. Whole plant habit.
2. Root system.
3. Leaf arrangement.
4. Leaf shape.
5. Flower structure.
6. Fruit or seed pod.
7. Storage organ where relevant.
8. Labeled callouts.

### Style Rules

1. Hand-drawn.
2. Clean linework.
3. Botanical-plate inspired.
4. Soft neutral palette.
5. Consistent labeling.

### Prompt Template

"Hand-drawn botanical plate of [plant], clean linework, whole plant habit, root system, leaf arrangement, flower or fruit close-up, labeled structures, modern botanical illustration style."

## 8. Quality And Verification Rules

### Maximum Accuracy Mode

1. No guessing.
2. No invented data.
3. All timing must match the Errington calendars.
4. All taxonomy must match authoritative botanical sources.
5. All uncertainty must be labeled.

### Conflict Resolution

1. Errington calendars override all external timing.
2. Botanical references override common-name ambiguity.
3. When uncertainty remains, use conservative and safe horticultural guidance.
